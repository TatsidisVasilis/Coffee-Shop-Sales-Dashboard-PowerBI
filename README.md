
# Coffee Shop Sales Dashboard (Power BI)

## About the Project

This is a Power BI portfolio project where I analysed coffee shop sales data and built an interactive dashboard to track sales performance, product trends and customer purchasing patterns.

The goal of the project was to practice Power BI, data visualisation and dashboard design using a real-world sales dataset.

## Tools Used

- Power BI
- Power Query
- DAX

## What I Analysed

- Total sales performance
- Total orders
- Total quantity sold
- Daily sales trends
- Sales by product category
- Sales by product type
- Sales by weekday vs weekend
- Sales by store location
- Sales by day and hour
- Monthly performance trends
